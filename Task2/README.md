##  Sum to n task
How to run:

To choose number to sum type -n 100, where instead of 100 type your number to sum

1. With number of threads from env

```
./build.sh
./run.sh -n 100
```

2. With typing number of threads

```
./build.sh
./run.sh -t 5 -n 100
```
Where instead of 5 type your number of threads
